# 🌍 World Life Expectancy Analysis

This project uses **MySQL** to clean and analyze a world life expectancy dataset. The analysis explores trends in life expectancy and its relationship with factors such as GDP, BMI, and development status.

## 📁 Project Files

- 📄 [Data_Cleaning.sql](./Data_Cleaning.sql)
- 📄 [EDA.sql](./EDA.sql)
- 📊 [WorldLifeExpectancyScript.sql](./WorldLifeExpectancyScript.sql)

## 📈 Analysis

### Data Cleaning
- Removed duplicate records.
- Populated missing values.

### Exploratory Data Analysis (EDA)
- Analyzed minimum, average, and maximum life expectancy by country.
- Identified countries with missing or incomplete data.
- Examined global life expectancy trends over time.
- Explored the relationship between GDP and life expectancy.
- Compared life expectancy between developed and developing countries.
- Investigated the impact of BMI on life expectancy.
